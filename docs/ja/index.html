JEditor


逆コンパイルと再変換/再定義を含む Java バイトコードエディタ

ぶんかい


Java バイトコードファイルを選択した場合,
JEditorはファイルを処理し、フォーマットで編集することができます。

再変換/再定義


プロセスを選択した後、
プロセスのすべてのクラスを表示し、編集することができます。

ダウンロード & 使用方法


ダウンロード後、JDK 8を使用して通常のJARファイルとして開きます。
もし何か問題があれば、 Github Issues を提出してください！
