JEditor


A java-bytecode editor including decompile and retransform/redefine

Decompile


When you select a java-bytecode file,
JEditor will process the file for you to edit by its format.

Hot retransform/redefine


After you select a process,
you can see all the classes of the process and then edit them.

Download & Usage


After you download it, use JDK 8 to open it as a normal JAR file.
If you have any problems, welcome to submit a Github Issues !
